oxford_dictionaries

oxford_dictionaries library Null safety

Dictionary class that uses the Oxford Dictionaries RESTful API.

Also exports the dictosaurus package.

Classes

AutoCorrect
The AutoCorrect interface exposes the suggestionsFor and startsWith methods.
AutoCorrectBase
An abstract class that implements the AutoCorrect interface.
AutoCorrectMixin
A mixin class that implements the AutoCorrect.suggestionsFor and AutoCorrect.startsWith methods.
Dictionary
The Dictionary interface exposes the getEntry method that returns a TermProperties for a term.
DictionaryBase
An abstract class with DictionaryMixin that implements the Dictionary interface.
DictionaryMixin
An abstract/mixin class that implements the definitionsFor, phrasesWith and inflectionsOf methods of Dictionary.
DictoSaurus
The DictoSaurus interface implements the Dictionary, Thesaurus and AutoCorrect interfaces.
DictoSaurusBase
Implements DictoSaurus by mixing in DictoSaurusMixin.
DictoSaurusMixin
A mixin class that implements DictoSaurus.expandTerm.
OxfordDictionaries
Implements Dictionary with the Oxford Dictionaries API as dictionary provider.
OxfordDictionariesApiMixin
A mixin that returns Map<String, dynamic> responses from the Oxford Dictionaries API.
TermDefinition
A definition for a defintion and/or part-of-speech variant of a term, used in a TermProperties object.
TermDefinitionBase
Abstract implementation class of TermDefinition that implements the == operator and hashCode.
TermDefinitionMixin
Abstract mixin class of TermDefinition that implements the == operator and hashCode.
TermProperties
An object model for a term or word with immutable properties.
TermPropertiesBase
An abstract base class with TermPropertiesMixin that implements the TermProperties interface.
TermPropertiesMixin
Abstract mixin class that implements the == operator and methods of the TermProperties interface.
Thesaurus
The Thesaurus interface exposes methods that return a term's synonymsOf and/or antonymsOf from a dictionary provider.
ThesaurusBase
An abstract class that implements the Thesaurus interface.
ThesaurusMixin
A mixin class that implements Thesaurus and uses getEntry to retrieve TermProperties from which it extracts synonymsOf and antonymsOf a term.
