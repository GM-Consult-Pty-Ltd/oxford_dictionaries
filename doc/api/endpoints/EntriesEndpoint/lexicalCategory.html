lexicalCategory

lexicalCategory property Null safety

PartOfSpeech? lexicalCategory
final

A comma-separated list of lexical categories ids to match on

Only term variants for the lexicalCategory are returned from the endpoint.

The available lexical categories for a given language (or language pair) can be obtained from the /lexicalcategories/ endpoint.

If null, variants for all parts of speech are returned.
